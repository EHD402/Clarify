# Clarity - Providing additional notes for pdf:s

This tool is allows you to upload pdf:s and provide notes for each specific page in that pdf. 

I primarily built this tool for fun and because it will be useful for me during my university courses. However, as I started to shift my focus onto other projects, this project was used as a testing ground for setting up and trying Claude code for the first time.

If requested or I feel the need for it, I may continue to develop this app and provide additional features + fix possible bugs.
